# KHOps Web Frontend

This folder contains the Next.js frontend for the KHOps platform. It is designed to deliver an enterprise-grade dashboard, modular components, and modern styling.

## Quick start

1. Navigate into the frontend workspace:
   ```bash
   cd web
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:3000` in your browser.

## Workspace structure

- `app/` — Next.js App Router pages and layout
- `components/` — Modular UI components
- `lib/` — frontend helpers and API stubs
- `types/` — shared TypeScript interfaces
- `tailwind.config.ts` — theme customization

## Notes

- The design is based on the provided dark enterprise dashboard inspiration.
- `core cluster` labels are neutral and not tied to AWS.
